Global Function API

  • isActivatableFloor(floorNum)

    Description
    층의 활성화 가능 여부를 확인하는 함수
    Arguments
    floorNum (Number): 몇층인지를 나타는 숫자값
    Returns
    (Boolean): 활성화가 가능하면 true 아니면 false(비활성화 된 floor는 true를 리턴함)
    Example
    let floorNum = 1; let result = isActivatableFloor(floorNum); if (result) { ... } else { ... }

  • activatingFloor(floorNum)

    Description
    층에서 엘레베이터를 호출하면 층은 비활성화가 되어 버튼이 클릭되지 않는다. 해당함수를 사용하면 비활성화가 풀리며 호출된 엘레베이터의 target floor는 undefined가된다.
    Arguments
    floorNum (Number): 몇층인지를 나타는 숫자값
    Returns
    (Object)
    • floorId: 활성화된 층의 아이디
    • elevatorId: 층에서 호출되어진 엘레베이터의 아이디
    Example
    let floorNum = 1; let result = isActivatableFloor(floorNum); let floorId = result.floorId; let elevatorId = result.elevatorId; ...
